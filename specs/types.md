# Система типов MyLang

## Встроенные типы

| Имя | Размер | Описание |
|-----|--------|----------|
| `int` | 64 бит | знаковое целое (псевдоним `int64`) |
| `uint` | 64 бит | беззнаковое целое (псевдоним `uint64`) |
| `int8` | 8 бит | знаковое целое |
| `int16` | 16 бит | знаковое целое |
| `int32` | 32 бит | знаковое целое |
| `int64` | 64 бит | знаковое целое |
| `uint8` | 8 бит | беззнаковое целое |
| `uint16` | 16 бит | беззнаковое целое |
| `uint32` | 32 бит | беззнаковое целое |
| `uint64` | 64 бит | беззнаковое целое |
| `float` | 64 бит | вещественное (псевдоним `float64`) |
| `float32` | 32 бит | вещественное (IEEE 754) |
| `float64` | 64 бит | вещественное (IEEE 754) |
| `bool` | 8 бит | булево значение (`true` / `false`) |
| `char` | 8 бит | ASCII-символ |
| `string` | — | строка (длина + байты) |
| `void` | — | отсутствие значения, только как тип возврата функции |

## Составные типы

### Массив

```
[T; N]
```

- `T` — тип элемента, `N` — целочисленный литерал (размер фиксирован)
- Размер является частью типа: `[int; 3]` и `[int; 4]` — разные типы
- Индексирование: `arr[i]`, индекс типа `int` или `uint`

### Структура

```
struct Point {
    int x;
    int y;
}
```

- Поля хранятся последовательно в памяти
- Доступ к полю: `p.x`
- Структуры передаются по значению (копирование)

## Номинальная типизация

Два типа совместимы только если их имена совпадают.

```
type Meters = int;
type Seconds = int;

Meters m = 10;
Seconds s = m;   // ОШИБКА: Meters != Seconds
```

Синоним типа — это альтернативное имя, а не новый тип:

```
type Meters = int;
int x = 10;
Meters m = x;   // OK: Meters это тот же int
```

## Нет неявных приведений

Любое преобразование типов явное через `cast<T>(expr)`:

```
int x = 5;
float y = cast<float>(x);   // OK
float z = x;                 // ОШИБКА
```

Допустимые приведения:

| От | К | Комментарий |
|----|---|-------------|
| любой int | любой int | narrowing/widening, усечение при narrowing |
| любой int | float32/float64 | расширение |
| float32/float64 | любой int | усечение дробной части |
| float32 | float64 | расширение |
| float64 | float32 | усечение |
| любой int | bool | 0 → false, остальное → true |
| bool | int | false → 0, true → 1 |
| char | int | значение байта |
| int | char | младший байт |

Приведения между `string` и числовыми типами не допускаются.

## Мутабельность

По умолчанию переменная мутабельна. `const` запрещает переприсваивание:

```
int x = 5;
x = 10;          // OK

const int MAX = 100;
MAX = 200;        // ОШИБКА: переменная константна
```

`const` на параметры функции не распространяется — параметры всегда мутабельны внутри тела функции (передача по значению).

## Тип числовых литералов

- Целый литерал (`123`) имеет тип `int`
- Вещественный литерал (`3.14`) имеет тип `float`
- При присваивании в переменную другого числового типа: совместимость проверяется статически, если тип переменной задан явно — приведение требуется явно

```
int x = 5;         // OK: 5 имеет тип int
int32 y = 5;       // OK: литерал совместим с int32
float z = 3.14;    // OK
int32 w = 3.14;    // ОШИБКА: float литерал в int переменную
```

## Строки

- Тип `string` — неизменяемая последовательность байт (UTF-8)
- Операции:
  - Конкатенация: `s1 + s2`
  - Длина: встроенная функция `len(s)` возвращает `int`
  - Сравнение: `==`, `!=`
- Индексирование строк не поддерживается

## Переполнение

Переполнение целых чисел — поведение как в C: для знаковых не определено (wraparound на x86-64), для беззнаковых — wraparound по модулю 2^N.

## Void

`void` допускается только как тип возврата функции. Переменных типа `void` не существует.
